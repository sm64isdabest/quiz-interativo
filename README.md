# TDE de Lógica de algoritmo.

## Sobre o Projeto
Este projeto consiste no desenvolvimento de um jogo educativo inspirado no Cuca Legal, com o objetivo de estimular o raciocínio lógico, o pensamento computacional e a resolução de problemas.

O jogo foi desenvolvido como atividade acadêmica para o curso de Engenharia de Software, com foco na aplicação prática de conceitos de lógica matemática e verificação de programas.

## Funcionalidades
- Sistema de perguntas com desafios lógicos

## Objetivo
O principal objetivo do jogo é tornar o aprendizado de lógica mais dinâmico e acessível, aproximando conceitos teóricos da prática de forma lúdica e interativa.

## Contexto Acadêmico
Projeto desenvolvido como parte de uma atividade proposta no curso de Engenharia de Software, orientado pelo Prof. M.Sc. Alberlan Lopes.
